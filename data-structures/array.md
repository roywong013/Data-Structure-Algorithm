# Array

## Dynamic Array

### Time Complexities
| Access  | Search | Insertion | Deletion | Append |
| ------- | -------| --------- | -------- |--------|
| O(1)    | O(n)   | O(n)      | O(n)     |O(1)*   |

<i>*can be O(n) if the size of original array do not have enough memory to append a new item. </i>
